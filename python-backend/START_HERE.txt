📊 SCORING SYSTEM (3 Dimensions)
══════════════════════════════════════════════════════════════════════════════

Score = (Clarity + Relevance + Correctness) / 3

1️⃣  CLARITY (Answer Structure)
     • Sentence structure & length
     • Vocabulary diversity
     • Logical flow & coherence
     • Completeness
     Range: 0.0 - 1.0

2️⃣  RELEVANCE (Question Alignment)
     • Question addressing
     • Role-specific keywords
     • Topic focus
     • On-topic responses
     Range: 0.0 - 1.0

3️⃣  CORRECTNESS (Technical Accuracy)
     • Technical accuracy (LLM-based)
     • Best practices validation
     • Industry standards
     • Common mistakes detection
     Range: 0.0 - 1.0

RESPONSE INCLUDES:
✓ Numerical scores for each dimension
✓ Total score (average of 3)
✓ Constructive feedback
✓ Strengths identification
✓ Improvement suggestions

🔌 INTEGRATION WITH NEXT.JS
══════════════════════════════════════════════════════════════════════════════

1. Add to .env.local:
   PYTHON_API_URL=http://localhost:8000

2. In your Next.js API route:
   const response = await fetch(`${process.env.PYTHON_API_URL}/api/score`, {
     method: 'POST',
     headers: { 'Content-Type': 'application/json' },
     body: JSON.stringify({
       transcript: userAnswer,
       question: currentQuestion,
       role,
       difficulty
     })
   })
   const scores = await response.json()

3. Use the response data in your frontend
