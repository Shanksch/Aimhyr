SCORING MECHANISM:
===================

Three-dimensional evaluation:

1. CLARITY (0-1)
   - Sentence structure & length optimization
   - Vocabulary diversity
   - Logical flow
   - Scoring: Too short (0.2-0.4) | Optimal (0.7-0.95)

2. RELEVANCE (0-1)
   - Question alignment check
   - Role-specific keywords
   - Topic focus
   - On-topic responses
   - Frontend: React, JavaScript, UI, CSS, component, state
   - Backend: API, database, SQL, scalability, deployment
   - Data Science: data, model, statistics, machine learning
   - DevOps: CI/CD, Docker, Kubernetes, infrastructure

3. CORRECTNESS (0-1)
   - Technical accuracy (LLM-based)
   - Best practices validation
   - Industry standards
   - Common mistakes detection
   - Scoring: 0.5 base + 0.08 per practice

TOTAL SCORE = Average(Clarity, Relevance, Correctness)


INTEGRATION WITH NEXT.JS:
=========================

Add to Next.js .env.local:
  PYTHON_API_URL=http://localhost:8000

Update Next.js route handler:
  const response = await fetch(`${process.env.PYTHON_API_URL}/api/score`, {
    method: 'POST',
    body: JSON.stringify({ transcript, question, role, difficulty })
  })
  const scores = await response.json()
